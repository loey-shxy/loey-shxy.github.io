### 配置参数 (Attributes)

| 参数名   | 描述   | 类型   | 默认值 |
| -------- | ------ | ------ | ------ |
| title    | 标题   | string | ''     |
| subTitle | 副标题 | string | ''     |

### 插槽

| 插槽名称    | 描述         | 参数 |
| ----------- | ------------ | ---- |
| title       | 标题         | -    |
| prefix-icon | 标题前置图标 | -    |
| suffix-icon | 标题后置图标 | -    |
| subTitle    | 副标题       | -    |
| extra       | 右侧操作组   | -    |
