### 配置参数 (Attributes)
| 参数名 | 描述     | 类型   | 默认值  |
| ------ | -------- | ------ | ------- |
| name   | 图标名   | string | ''      |
| size   | 图标大小 | number | 1em     |
| color  | 图标颜色 | string | #4E5969 |
