### 2、配置参数（Attributes）

| 参数            | 说明        | 类型    | 默认值    |
| :-------------- | :---------- | :------ | :-------- |
| width           | 图片宽度    | String  | 2.0408rem |
| height          | 图片高度    | String  | 2.0408rem |
| lazyLoad        | 懒加载      | Boolean | false     |
| arco.design参数 | a-image一样 |         |           |
